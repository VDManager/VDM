# VDM

Virtual Desktop Manager

With this app you can configure virtual desktop presets, that you can launch on-demand or on startup automatically.

## Requirements

- .NET Core 3.0
- Windows 10

## How to build

```
dotnet restore
dotnet build
```

Enjoy!
